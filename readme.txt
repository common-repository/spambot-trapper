Spambot Trapper stops most spambot-posted comments from ever reaching your spam queue.

== Description ==

Tired of slogging through automated spam? Spambot Trapper stops most spambot-posted comments from ever reaching your spam queue by changing comment form field names and adding invisible "trap" fields. Spambot Trapper is an easy and lightweight way to stop those evil spambots dead in their tracks.

To start using this plugin, just install, activate, and say goodbye to automated spam!

If you'd like, you can even change the `name` attributes of the different comment fields in the WordPress Discussion Settings.

== Installation ==

1. Upload the `/spambot-trapper/` directory to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
